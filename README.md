<h1><img src="https://raw.githubusercontent.com/duboviy/web/master/logo.png" height=85 alt="logo" title="logo"> web</h1>
A sandbox for trying out web frameworks:

* Django
* Sinatra (Ruby)
* Flask
* NodeJS
* NodeJS + React
* Ruby on Rails
